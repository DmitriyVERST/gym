# Полный Ежедневный Тренировка Дома

# Live: https://dmitriyverst.github.io/gym/

Этот проект представляет собой комплексный план ежедневных тренировок, которые можно выполнять дома без специального оборудования. Программа разработана для укрепления мышц, повышения выносливости и улучшения гибкости. Подходит для всех уровней подготовки — от новичков до продвинутых спортсменов.

## Как использовать
Просто следуйте ежедневному плану, выполняя упражнения по указанной последовательности. Регулярность — ключ к успеху!

## Установка и запуск
1. Склонируйте репозиторий

``` bash
git clone https://github.com/DmitriyVERST/gym.git
```  
2. Откройте `index.html` в браузере
